1. Positional Numeral Systems — Overview

  • In a positional (place value) system, the position of a digit determines its value.
  • The Babylonian system (base 60) was the first positional system — its influence survives in 60 minutes/hour and 360° in a circle.
  • The modern positional system, including zero as a placeholder, was developed in India around the 7th century.
  • A base (radix) is the number of digits a system uses. Decimal = base 10 (digits 0–9).
  • A subscript distinguishes bases, e.g. 1002 (binary) ≠ 10010 (decimal) ≠ 10016 (hex).
  • Binary (base 2) is used in computers because it's easy to implement electronically. Octal (base 8) and hexadecimal (base 16) represent binary values more compactly for humans (e.g. memory addresses, web colour codes).

3. Quinary (Base-5) System

  • Quinary is a base-5 numeral system — possibly originating from counting on the five fingers of one hand.
  • Uses five numerals: 0, 1, 2, 3, 4. Each place value is a power of 5.
Position...6th5th4th3rd2nd1st
Place value...5⁵ = 3,1255⁴ = 6255³ = 1255² = 255¹ = 51

4. Binary (Base-2) System

  • Binary uses only two digits: 0 and 1. It is the foundation of all digital/computing systems.
  • Each place value is a power of 2, increasing right to left.
Position...6th5th4th3rd2nd1st
Place value...2⁵ = 322⁴ = 162³ = 82² = 42¹ = 21

Worked Example — 11001₂ and 101110₂

11001₂: digits are 1(16) 1(8) 0(4) 0(2) 1(1) → 16 + 8 + 0 + 0 + 1 = 25
101110₂: (1×2⁵)+(0×2⁴)+(1×2³)+(1×2²)+(1×2¹)+(0×2⁰) = 32+0+8+4+2+0 = 46

5. Roman Numerals

  • Originated in ancient Rome; used in Europe well into the late Middle Ages. Still seen today on clock faces, monument dates, book chapters, and film/TV copyright dates.
  • Seven main symbols:
IVXLCDM
1510501005001000
  • Addition when a smaller symbol follows a larger one; subtraction when a smaller symbol precedes a larger one.
  • Only I, X, C can precede another symbol to indicate subtraction: I before V or X only; X before L or C only; C before D or M only.
  • A symbol cannot repeat more than three times in a row. Only one symbol may precede another (no double-subtraction). V, L, D are never repeated.
  • A bar (vinculum) over a numeral multiplies its value by 1,000.
  • There is no symbol for zero in the Roman system.

Q5 — Word problem

Tae has V̄LVI cookies. Tarn has X̄MV̄DXLIV cookies. If they combine their cookies and pack them into boxes of LXXX (80) cookies each, how many boxes will they have?

  • Tae: V̄LVI = 5,000 + 56 = 5,056 cookies
  • Tarn: X̄MV̄DXLIV = 10,000 + 1,000 + 500 + 40 + 4 = 11,544 cookies (read left to right as shown in the text — please confirm bar placement against the original)
  • Total = 5,056 + 11,544 = 16,600 cookies
  • LXXX = 80 cookies per box
  • 16,600 ÷ 80 = 207.5 → 207 full boxes (with 40 cookies left over)
Note: Tarn's numeral (X̄MV̄DXLIV) contains an unusual bar placement that was difficult to confirm from the photo. Please double-check the original page — if the numeral differs, only the total and final division step need to be recalculated.
